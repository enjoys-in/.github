## 👋 About Us

**enjoys.in** is an indie engineering org building **production-grade, self-hostable infrastructure** — email servers, voice/VoIP platforms, calendar backends, and React developer plugins. We love TypeScript, Go, and Rust, and we ship things we actually run ourselves.

> Think open-source alternatives to the SaaS tools you pay for — fully programmable and yours to host.

---

## 🚀 What We're Building

| Project | Description | Stack |
| --- | --- | --- |
| [**enjoys-voice**](https://github.com/enjoys-in/enjoys-voice) | Self-hosted voice / VoIP platform with anti-fraud, multi-party conferences & call queues | `TypeScript` `Go` `FreeSWITCH` |
| [**react-chatbot-plugin**](https://github.com/enjoys-in/react-chatbot-plugin) | Fully customizable chatbot widget for React — JSON flows, 15+ form fields, plugins. Like tawk.to, but open-source | `TypeScript` `React` |
| [**server-stack-suite-backend**](https://github.com/enjoys-in/server-stack-suite-backend) | All-in-one self-hostable server management suite | `TypeScript` |
| [**cal-dev**](https://github.com/enjoys-in/cal-dev) | Calendar server (Go + Fiber + PostgreSQL + Valkey) with CalDAV / JMAP & Google/Apple interop | `Go` |
| [**gluon**](https://github.com/enjoys-in/gluon) | IMAP server library in Go | `Go` |
| [**keypal**](https://github.com/enjoys-in/keypal) | Secure API key management — hashing, scopes, expiry, pluggable storage | `TypeScript` |
| [**react-api**](https://github.com/enjoys-in/react-api) | Strongly-typed storage utility using CacheStorage & IndexedDB with sync events | `TypeScript` |
| [**mailbridge**](https://github.com/enjoys-in/mailbridge) | Mail routing & bridging service | `TypeScript` |

➡️ [**Browse all repositories**](https://github.com/orgs/enjoys-in/repositories)
